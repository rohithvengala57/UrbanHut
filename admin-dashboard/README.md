# UrbanHut Admin Dashboard

A centralized management interface for the UrbanHut platform, built with React and Vite.

## Features

### Phase 1: Executive Overview (Implemented)
- Top KPI Cards (Users, Listings, Interests, Messages)
- Activity Timeline
- Feature Usage & Growth Placeholders

### Phase 2: Product Analytics (Implemented)
- **Marketplace Funnels**: Onboarding and Seeker journey conversion visualizations.
- **Retention Analytics**: Cohort analysis table and D1/D7/D30 retention trends.
- **Search Analytics**: Top cities distribution and filter usage insights.
- **Design System Integration**: Consistent UI based on `admin/DESIGN_SYSTEM.md`.

## Tech Stack
- **Framework**: React 19
- **Build Tool**: Vite 8
- **Visualization**: Recharts
- **Icons**: Lucide-React
- **Routing**: React Router 7

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Design System
Refer to `admin/DESIGN_SYSTEM.md` for color palettes, typography, and layout constants.
